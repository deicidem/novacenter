Примечания

  • программа IMAGE MEDIA CENTER работает только на компьютере, на котором был зарегистрирован электронный ключ защиты;
  • для работы модулей необходимо наличие лицензии одной из версий IMAGE MEDIA CENTER;
  • стоимость указана для одного рабочего места / сервера. НДС не облагается;
  • срок функционирования модулей равен сроку действия лицензии IMAGE MEDIA CENTER;
  • в стоимость лицензии не входит стоимость технической поддержки;
  • техническая поддержка включает в себя предоставление обновлений, консультации по поставляемому программному обеспечению посредством обмена электронными сообщениями, по телефону или в офисе ООО «Центр инновационных технологий»;
  • переход с версии Professional на Enterprise с тем же составом модулей производится по разнице стоимостей соответствующих версий;
  • предоставление обновлений, переход с версии Professional на Enterprise, добавление модулей производятся только при действующей технической поддержке;
  • обновление программы осуществяляется через Интернет или почтой на компакт-диске.
